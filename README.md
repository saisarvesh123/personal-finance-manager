# Personal Finance Manager

A Java-based application designed to help users manage their personal finances by tracking expenses, managing budgets, and maintaining transaction records using file handling.

## Features

* User Authentication
* Expense and Income Tracking
* Budget Management
* Transaction History
* File-Based Data Storage

## Technologies Used

* Java
* IntelliJ IDEA
* Object-Oriented Programming (OOP)
* File Handling

## How to Run

1. Open the project in IntelliJ IDEA.
2. Configure the JDK.
3. Run `MainApp.java`.
